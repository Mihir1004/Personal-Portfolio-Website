# Mihir Portfolio Website

This is a static one-page portfolio website built with plain HTML, CSS, and JavaScript.
It is ready to deploy on Vercel without any build step.

## Project structure

- `index.html` contains the page content
- `assets/css/styles.css` contains all styles
- `assets/js/main.js` contains navigation, active link, and reveal interactions
- `vercel.json` contains basic Vercel deployment settings

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. Sign in to Vercel and click `Add New -> Project`.
3. Import the GitHub repository.
4. Keep the default settings because this is a static site.
5. Deploy.

## Customize before publishing

- Replace the email address in `index.html`
- Replace the social profile links in `index.html`
- Update the project titles, descriptions, and stats
- Change the images in `assets/img/` if needed
